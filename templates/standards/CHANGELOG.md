# Standards Changelog

Track all promotions from `.chaos/learnings.md` into `standards/`.

## Format

Each entry records:
- **Date**: When the promotion happened
- **Source**: Task ID or session that generated the observations
- **Evidence**: Number of occurrences that triggered promotion
- **Target**: Which standards file was updated
- **Pattern**: Brief description of what was promoted
